Utility Transform Inspector (UTI)

The UTI web application provides a simple interface to inspect and identify interval meter data. The front-end of the application uses R-Shiny and base R with data manipulation using the data.table library visualizations are acheived via plotly. Data is sourced from data retrieval and transformation pipeline, since different utilities provide customer data in a host of different formats standardization routines try to best collate and normalize this data. No process is perfect hence the UTI introduces the human component to verify proper transformation received data.
Once a data packet has been approved by an Analyst the data is normalized to be stored in a PostGRE database hosted on AWS
